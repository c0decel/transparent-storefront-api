Products

productRoutes.js

Business Logic URL Method Query Params Permissions Response
Get all products /products GET None None JSON of all products
Get one product /products/:id GET ID of product None JSON of one product
Get tags from one product /products/:id/tags GET ID of product None Array of product's tags
Get product by tag /products/tags/:id GET ID of product None Array of products containing searched tag
Get supplies from one product /products/:id/supplies GET ID of supply None Array of product's supplies
Post new product /products POST None Admin JSON of new product
Create discount code /products/:id POST None Admin JSON of new discount
Update stock quantity /products/:id PUT ID of product Admin JSON of new stock quantity
Add tag to product /products/:id/tags/:tagId PUT ID of product and ID of tag Admin Array of product's tags
Add supply to product /products/:id/supplies/:supplyId PUT ID of product and ID of supply Admin Array of product's supplies
Remove tag from product /products/:id/tags/:tagId DELETE ID of product and ID of tag Admin Array of product's tags
Remove supply from product /products/:id/supplies/:supplyId DELETE ID of product and ID of supply Admin Array of product's supplies
Delete a product /products/:id DELETE ID of product Admin Deletes a product
