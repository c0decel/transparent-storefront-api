threadRoutes.js

Business Logic URL Method Query Params Permissions Response
Get all threads /threads GET None None JSON of all thread OP posts
Get one thread /threads/:id GET ID of thread None JSON of one thread OP post
Get replies from one thread /threads/:id/replies GET ID of thread None JSON of all responses to one thread
Post new thread OP /threads POST None Logged in JSON of new thread OP
Get threads by tag /threads/tags/:id GET ID of tag to search Logged in JSON of threads including searched tag
Delete a thread /threads/:id DELETE ID of thread Admin Deletes thread
Threadban a user /threads/:id/bans/:Username/:Post POST ID of thread, username of user to ban, ID of post to cite in ban Admin Bans user from thread until specified expiration date
